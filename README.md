# Task Manager Web Application

## Project Overview
This is a full-stack task management application where users can register, login, and manage their tasks. Users can create, view, update, and delete tasks.

## Tech Stack Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Features Implemented
- User Registration and Login
- Create Task
- View Tasks
- Update Task Status
- Delete Task

## Setup Instructions
1. Install Node.js
2. Download or clone this repository
3. Open terminal and go to backend folder:
   cd backend
4. Install dependencies:
   npm install
5. Start server:
   node server.js
6. Open frontend/index.html in browser
